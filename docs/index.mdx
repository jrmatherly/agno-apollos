title: "Apollos AI"
description: "A multi-agent system on Docker, powered by the Agno framework."
---

Apollos AI is a self-hosted, containerized platform for running multiple AI agents behind a single FastAPI backend. It uses [Agno](https://docs.agno.com) for agent orchestration, PostgreSQL with pgvector for storage and retrieval, and a Next.js frontend for the control plane UI.

## What's included

### Agents

| Agent | Pattern | Description |
|-------|---------|-------------|
| Knowledge Agent | Agentic RAG | Hybrid search over a vector knowledge base with learning and user profiles |
| MCP Agent | MCP Tool Use | Connects to external services via the Model Context Protocol |
| Web Search Agent | Web Research | Searches the web via DuckDuckGo with source citations |
| Reasoning Agent | Chain-of-Thought | Step-by-step reasoning with configurable depth |
| Data Analyst | SQL Analysis | Read-only PostgreSQL queries following the Dash pattern |
| M365 Agent | MCP Tool Use | Read-only Microsoft 365 access (opt-in) |

### Teams and workflows

| Name | Type | Description |
|------|------|-------------|
| Coordinator | Coordinate team | Intelligent routing to the best agent for each query |
| Research Team | Coordinate team | Multi-agent research combining web search and analysis |
| Research Workflow | Pipeline | Automated web search to structured report |

## Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python 3.12, FastAPI, Agno framework |
| Frontend | Next.js 15, React 18, TypeScript, Tailwind CSS, shadcn/ui |
| Database | PostgreSQL 18 + pgvector |
| LLM routing | LiteLLM Proxy (any provider) |
| Tooling | mise (task runner), uv (Python), pnpm (frontend) |
| Deployment | Docker Compose (dev + prod configurations) |
